Purpose: to demonstrate how to use the petimer.
Instruction:
    1. Run the ./sample_petimer count command.
    2. Count is the number of timers. The value of count must be greater than 2000 during test. These timers can be released automatically.
    3. When this demonstration starts, the number of enabled timers and the number of released timers are displayed. You can check whether the timers are released automatically by viewing the displayed information.
