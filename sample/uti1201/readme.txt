Purpose: to demonstrate the DVB playing scenario using uti.
Command: sample_uti1201 freq srate [qamtype] [vo_format]
      Fields qamtype and vo_format are optional.
      The value of qamtype can be 16, 32, 64, 128, 256, or 512. The default value is 64 QAM.
      The value of vo_format can be 1080P_60, 1080P_50, 1080i_60, 1080i_50, 720P_60, or 720P_50. The default value is 1080i_60.
