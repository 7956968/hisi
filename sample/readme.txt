Sample Content Description    
    3d---------------------------------Demonstration of three-dimensional (3D) functions
    aiao-------------------------------Demonstration of audio input and output
    capture----------------------------Demonstration of screen shot   
    common-----------------------------Demonstration of common parameters
    demux------------------------------Demonstration of filter function
    dvbplay----------------------------Demonstration of playing in digital video broadcasting (DVB) mode
    ecs--------------------------------Demonstration of peripherals
    esplay-----------------------------Demonstration of playing local element stream (ES) files.
    fb---------------------------------Demonstration of functions provide by HiFB
    flash------------------------------Demonstration of flash operation cases
    fs---------------------------------Demonstration of the file system
    ha_libmad--------------------------Demonstration of registering the audio decoding library
    hdmi_tsplay------------------------Demonstration of high definition multimedia interface (HDMI)
    higo-------------------------------Demonstration of how to use HIGO
    iframe_dec-------------------------Demonstration of decoding I frames
    ipplay-----------------------------Demonstration of playing in IP mode
    mix--------------------------------Demonstration of audio mixing
    mosaic-----------------------------Demonstration of playing in mosaic mode
    mp3--------------------------------Demonstration of displaying mp3 spectrum diagram 
    network----------------------------Demonstration of network-related tools
    petimer----------------------------Demonstration of how to use petimer
    pip--------------------------------Demonstration of picture-in-picture (PIP) playing
    psi--------------------------------Demonstration of the search function
    pvr_rec----------------------------Demonstration of recording function of the PVR module
    pvr_play---------------------------Demonstration of playback function of the PVR module
    pvr_timeshift----------------------Demonstration of time-shift playing of the PVR module
    pvr_rec_allts----------------------Demonstration of recording streams of the PVR module
    pvr_sample-------------------------Demonstration of search, live broadcast, recording, and playback
    res--------------------------------Demonstration of resource files
    tsplay-----------------------------Demonstration of playing local transport stream (TS) files.
    tuner------------------------------Demonstration of locking frequencies
    uti1201----------------------------Demonstration of using uti chip 1201
    es_ts_switch-----------------------Demonstration of switch between playing local ES files and TS files.

For details about how to use samples, see the readme.txt file in each directory.
