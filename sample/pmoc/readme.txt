Purpose: to demonstrate how to use the interface related to standby.

Instruction:
./sample_pmoc
Set the following parameters based on the displayed information:
	Whether to set the wakeup mode and timeout wakeup value
	LED display in standby mode: no display/channel/time 
	Number of retry times in standby mode

./sample_auto_pmoc
The stb will auto standby when detecting TV is closed or standby.
The return information contains the wakeup mode.
